Edit Mode Manual

このページは edit モードの全体像と、配下にある各ボタンの役割をまとめた説明書です。 実際の UI ツリーに合わせて、rail、creat、Difference の順で整理しています。

アプリを開く edit.html を開く
root: see / edit
edit 配下: rail / creat
対象: ボタン操作の意味

最初に把握しておくこと

  • see は鑑賞モード、edit は編集系モードの入口です。
  • edit に入ると、編集対象が rail と creat に分かれます。
  • rail は線路と線路由来の構造物向け、creat は自由配置のモデリング向けです。
  • ボタンを押すと配下が展開され、同時にそのモードに対応した内部処理が有効になります。
[ 重要 : 編集内容の保存について ]
/// 編集内容が消失するのを防ぐため、必ずご確認ください。///

編集内容の保存は右上の「ワールド保存」ボタンから行うことができます。

ボタンが押されると、編集内容が保存された zip ( 構造物 [ st_~ ] ) ファイルとなり、
お使いのタブレットやPCにダウンロードされ保存されます。

現在は、自動保存や、サービス内で保存するような機能はありません。
(公開する際はサービス内で保存されます。)
編集内容が消失するのを防ぐため、こまめな保存を推奨します。

Edit モードの構造

edit
🛠️

edit

編集モード全体の入口です。ここを押すと、配下の rail と creat が使える状態になります。

  • rail: 線路の制御点、線路移動、線路ベースの構造生成を行います。
  • creat: 点を打って形状を作り、装飾や部材、Difference 系編集を行います。

rail 系のボタン

edit > rail
🛤️

rail

線路編集モードです。線路制御点を対象にした追加・移動・構造生成が有効になります。

edit > rail > new

new

線路の制御点を追加します。直線ドラフトや新しいレールの起点づくりに使います。

edit > rail > move

move

既存の線路制御点を移動するモードです。
線路の形状を変更できます。

⤮ x_z
⇡ y

x_z

X/Z 平面で移動します。地面に沿った位置合わせ向けです。

y

Y 軸方向だけ動かします。高さだけを変えたいときに使います。

edit > rail > structure

structure

線路に関連する構造物を扱う枝です。配置ピンを打ってからカテゴリ生成します。

+new
construction

new

構造物配置用のピンを追加します。橋脚や壁などを生成する際の対象区間を設定するために使用します。

creat 系のボタン

edit > creat

creat

creatモード・簡易モデリングツールの入り口です。
これは点と点を結ぶ形で構造を作成する方式のツールで、
駅や橋など鉄骨感のある表現を簡単に再現することができます。

edit > creat > view

view

クリエイト対象の表示確認用です。通常表示に戻しながら、点やコピー対象の見え方を整理します。

edit > creat > decoration

decoration

モデリングツールの実際の作業をまとめた枝です。

train panel / utility buttons

edit 中は右側に寄せた補助ボタン群として、夜景切替、列車設定、ground 透過が使えます。

Rail Construction Category

ピンを選択し、カテゴリを選んで生成します。

Difference Category

spaceで平面を配置すると半透明プレビューを表示します。

rail 試験: 2本の線路にピンを打ち、指定して構造物を生成

軽量版です。線路をクリックすると近い位置にピンを打ちます。打ったピンをクリックして選択し、カテゴリを選んで生成すると、そのピン列に沿って簡易構造物プレビューを出します。

使い分けの目安

  • 線路を編集したいときは edit > rail を使います。
  • 駅、柱、板、装飾物のような自由配置物は edit > creat を使います。
  • 物体を削るための作業は edit > creat > Difference を使います。
  • 回転や移動は対象ごとに枝が分かれているので、rail と creat を混同しない方が安全です。

編集モードの説明

このモードでは、線路の形状の編集、構造物の配置から作成まで行えます。
以下は edit と creat で分けた説明です。

モード別デモ動画（見出しクリックで再生）

見出しをクリックすると動画を再生します。

モード早見表（UIボタン別）

  • 🎦 see: 鑑賞モード。編集用の当たり判定を無効化し、表示確認に集中します。
  • 🛠️ edit: 編集モードの入口。以下の rail / creat / custom を選んで作業します。
  • 🛤️ rail: 線路編集モード。
  • rail → new: 線路の制御点を追加します。
  • rail → move: 線路制御点を移動します。
  • move → x_z / y: 平面移動か高さ移動かを切り替えます。
  • rail → structure → new: 構造物配置用のピンを追加します。
  • rail → structure → construction: ピンから bridge / elevated / wall / floor / pillar / rib_bridge / tunnel_rect を生成します。
  • creat → view: クリエイト対象の表示/非表示を確認します。
  • creat → add_point: 点を追加して形状の基礎を作ります。
  • add_point → x_z_move: 追加基準面を平面（XZ）方向に移動します。
  • add_point → y_add: 追加基準面の高さ（Y）を調整します。
  • add_point → template: テンプレートガイドを使って配置します。
  • add_point → guide → add: ガイド面を追加します。
  • add_point → guide → change_angle: 追加したガイド面の角度を調整します。
  • creat → rotation: 選択点群をまとめて回転します（回転パネル対応）。
  • creat → move_point: 既存点を直接移動します。
  • move_point → x_z_sf / y_sf: 点移動の軸を切り替えます。
  • move_point → rotation: 点ごとの回転ギズモを使って向きを編集します（回転パネル対応）。
  • creat → construction: 点列から部材を生成します。
  • construction → pillar → Round_bar / H_beam: 柱部材の断面種別を選択して生成します。
  • construction → rite → tubular: チューブ部材を生成します。
  • custom → new / move / construct: カスタム形状向けの追加・移動・構築を行います。

edit（編集）/ 軌道の編集や、橋・トンネルといった構造物の生成を簡単に行えるモード

  • 🛠️ edit: 編集モードの開始。以降のタップ/クリックは編集動作になります。
  • 🛤️ rail: 軌道編集モード。`new` で制御点追加、`move` で点移動、`x_z`/`y` で移動平面を切替。
  • 🌉 structure → new: 軌道上にピンを追加。
  • 🏗️ construction: 追加したピンを選択して bridge / elevated / wall / floor / pillar / rib_bridge を生成。
  • 軌道保存 / ピン保存: 現在の軌道・ピン配置を保存します。

creat（作成）/ 鉄骨や板材、電光掲示板などを自由に配置し、駅や構築物をモデリングするモード

  • creat → view: クリエイト系の表示切替。
  • creat → add_point: 点の追加モード。タップで点を生成し、軌道として結びます。
  • add_point → x_z_move: 追加基準面を平面（XZ）方向に移動する補助モード。
  • add_point → y_add: 高さ方向（Y）だけを移動する補助モード。
  • add_point → template: ガイド棚を表示。S/L/U/直線のガイド軌道を配置できます。
  • add_point → guide → add: ガイド棚を表示。S/L/U/直線のガイド軌道を配置できます。
  • template / add のホバー: ガイド軌道にホバー中はガイド色が変化し、ピンが表示されます。
  • add_point のガイド配置: ガイド軌道に当たっている間は、その軌道上に点を打てます。
  • creat → construction: 追加した点を選択して `pillar` / `rite` などの部材を選択して生成。
  • move_point: 既存の点を移動。`x_z_sf`/`y_sf` で移動平面切替。
  • rotation: 複数点を選択すると中心にギズモ表示。リングドラッグで点の位置だけ回転。
  • rotation panel: X/Y/Z の角度を入力して絶対角度で回転。表示は現在角度を薄く表示。

ポイント: `add_point` を有効にするとガイド判定やガイドチューブが有効になります。`move` 中はガイドホバーは停止します。

※ モバイルではタッチ操作、デスクトップではマウスで操作できます。

### ⚒️ 編集モード 詳細
パソコンのパネル,マウスで **スクリーン** を **クリック&ドラッグ** でマップ上に物体を生成、又は位置を編集。<br>
物体を生成する場合は何もない場所をクリック。(生成時は平面上の座標を参照)

> 📦<br>🔺

物体を移動させる場合は、物体又は中間点にカーソルを合わせ、目的の場所へドラッグすることで移動可能。<br>

>⏥ >>>>>>>>> 📦<br>🔺 >>>>>>>>> 🔺
